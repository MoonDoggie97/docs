<Steps>
  <Step title="Important USB Requirement">
    ## ⚠️ USB Requirement (EFT Mason Full)

    - The loader **MUST** be launched from a **USB flash drive**  
    - The loader file must **NOT remain on your PC**  
    - Ensure the loader exists **ONLY on the USB drive**  
    - After a successful injection, **remove the USB immediately**  

  </Step>

  <Step title="Game Preparation">
    ## 1️⃣ Prepare the Game

    1️⃣ Launch the game  
    2️⃣ Set **Window Mode** to **"Fullscreen in Window"**  
    3️⃣ Close the game completely  

  </Step>

  <Step title="Loader Setup">
    ## 2️⃣ Download & Setup Loader

    1️⃣ Download the loader (link provided after purchase)  
    2️⃣ Use your license key  
    3️⃣ Move the loader file to your **USB flash drive**  

  </Step>

  <Step title="First Injection">
    ## 3️⃣ First Injection

    1️⃣ Run the loader as **Administrator** (from USB)  
    2️⃣ Select the game  
    3️⃣ Click **Run**  

    - Wait for the loader to complete  

    **If successful:**
    - A checkmark message will appear  
    - The loader will close automatically  
    - You will be prompted to restart → click **Yes**  

  </Step>

  <Step title="Second Injection">
    ## 4️⃣ Second Injection (After Restart)

    1️⃣ Run the loader again as **Administrator** (from USB)  
    2️⃣ Select the game and click **Run**  
    3️⃣ When prompted:  
       - Choose whether to **hide overlay**  

    - The loader will close automatically  
    - **Remove the USB flash drive immediately**  
    - Launch the game **yourself right away**  

  </Step>

  <Step title="In-Game Usage">
    ## 5️⃣ In-Game

    - Wait until the game fully loads to the main menu  
    - Press **HOME** to open the cheat menu  

  </Step>

  <Step title="HWID Spoofer">

  <Step title="Config System">
    ## 7️⃣ Saving & Loading Config

    ### Saving Config
    1️⃣ Open cheat menu  
    2️⃣ Go to **CONFIGURATION**  
    3️⃣ Click **Save config to clipboard**  
    4️⃣ Minimize the game  
    5️⃣ Paste into a new **Notepad file**  
    6️⃣ Save the file  

    ### Loading Config
    1️⃣ Launch cheat and game  
    2️⃣ Minimize the game  
    3️⃣ Open saved Notepad file  
    4️⃣ Press **CTRL + A**, then **Copy**  
    5️⃣ Return to the game  
    6️⃣ Open **CONFIGURATION**  
    7️⃣ Click **Load config from clipboard**  

  </Step>
