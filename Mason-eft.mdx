<Steps>
  <Step title="Important USB Requirement">
    ## ⚠️ USB Requirement (EFT Mason Full)

    - The loader **MUST** be launched from a **USB flash drive**
    - The loader file must **NOT remain on your PC**
    - Ensure the loader exists **ONLY on the USB drive**
    - After a successful injection, **remove the USB immediately**
  </Step>
  <Step title="Game Preparation">
    ##  Prepare the Game

     Launch the game\
     Set **Window Mode** to **"Fullscreen in Window"**\
     Close the game completely
  </Step>
  <Step title="Loader Setup">
    ##  Download & Setup Loader

     Download the loader (link provided after purchase)\
     Use your license key\
     Move the loader file to your **USB flash drive**
  </Step>
  <Step title="First Injection">
    ##  First Injection

     Run the loader as **Administrator** (from USB)\
     Select the game\
     Click **Run**

    - Wait for the loader to complete

    **If successful:**

    - A checkmark message will appear
    - The loader will close automatically
    - You will be prompted to restart → click **Yes**
  </Step>
  <Step title="Second Injection">
    ##  Second Injection (After Restart)

     Run the loader again as **Administrator** (from USB)\
     Select the game and click **Run**\
     When prompted:

    - Choose whether to **hide overlay**
    - The loader will close automatically
    - **Remove the USB flash drive immediately**
    - Launch the game **yourself right away**
  </Step>
  <Step title="In-Game Usage">
    ##  In-Game

    - Wait until the game fully loads to the main menu
    - Press **HOME** to open the cheat menu
  </Step>
  <Step title="HWID Spoofer">
    ##  HWID Spoofer

    - The HWID Spoofer is **integrated** into the loader

    After restarting your PC and launching the loader:

    - You may see a prompt asking to launch the spoofer
    - Click **Yes** if needed and follow instructions

    ### Verification (Optional)

     Download: [https://disk.yandex.ru/d/p4EhCbISg2Z2bg](https://disk.yandex.ru/d/p4EhCbISg2Z2bg)\
     Run and take a screenshot of current serials\
     Launch the HWID Spoofer via loader\
     Run the tool again before launching the game

    - If serials changed → spoofing worked
  </Step>
  <Step title="Config System">
    ##  Saving & Loading Config

    ### Saving Config

     Open cheat menu\
     Go to **CONFIGURATION**\
     Click **Save config to clipboard**\
     Minimize the game\
     Paste into a new **Notepad file**\
     Save the file

    ### Loading Config

     Launch cheat and game\
     Minimize the game\
     Open saved Notepad file\
     Press **CTRL \+ A**, then **Copy**\
     Return to the game\
     Open **CONFIGURATION**\
     Click **Load config from clipboard**
  </Step>
</Steps>
