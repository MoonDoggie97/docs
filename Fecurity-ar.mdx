<Steps>
  <Step title="Activating & Using the Cheat">
    ## Setup & Usage

    ### 1️⃣ Run the Loader as Administrator
    - Right-click the loader file  
    - Select **Run as administrator**

    ### 2️⃣ Enter Your Activation Key
    - When prompted, insert your activation key and press **Enter**  
    - A message will display your remaining access time and further instructions

    ### 3️⃣ Restart Your Computer
    - If you see: **"After Restarting Computer, launch the game"**, the setup was successful  
    - Your PC will restart automatically within a few seconds  

    ### 4️⃣ Launch the Game
    - After restarting, open the game normally  
    - A notification will confirm that the cheat has started successfully  

    ### 5️⃣ Using the Cheat
    - Press the **Delete** key to open/close the cheat menu  

    ### 6️⃣ After Playing
    - Restart your PC after you're done playing  
    - This helps prevent interference with other applications  

  </Step>

  <Step title="Troubleshooting">
    ## ⚠️ What to Do If the Cheat Does Not Work

    ### 1️⃣ Disable Windows Defender & Antivirus
    - The most common issue is antivirus software blocking the cheat  
    - Completely disable **Windows Defender (Real-time Protection)**  
    - Uninstall any third-party antivirus software to prevent conflicts  

    ### 2️⃣ Remove Conflicting Anti-Cheat Software
    - **Faceit AC** and **Riot Vanguard (Valorant Anti-Cheat)** must be uninstalled  
    - Use proper **software uninstalling**, not just deleting folders or shortcuts  

    ### 3️⃣ Check BIOS Boot Mode (UEFI Required)
    - The cheat will not work if your Boot Mode is set to **Legacy**  

    **To check your Boot Mode:**
    1️⃣ Press **Win + R** and type `msinfo32`  
    2️⃣ Look for **"BIOS Mode"** – it should be **UEFI**  

    - If Legacy mode is enabled, switch it to **UEFI** in your BIOS settings  

    ### 4️⃣ Disable Secure Boot
    - Secure Boot must be turned **off**  

    **To check Secure Boot status:**
    1️⃣ Press **Win + R** and type `msinfo32`  
    2️⃣ Look for **"Secure Boot Status"** – it should say **Off**  

    - If it's enabled, disable it in your BIOS settings  

  </Step>
</Steps>
