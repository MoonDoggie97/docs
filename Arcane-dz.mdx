  <Step title="Initial Setup">
    ## Pre-Setup (Before Key Activation)

    ### Required Installation

    - Install Microsoft Visual C\+\+ Redistributable:\
      [https://aka.ms/vs/17/release/vc\_redist.x64.exe](https://aka.ms/vs/17/release/vc_redist.x64.exe)
    - Restart your PC after installation (**mandatory**)

    ### Disable Conflicting Software

    Make sure you are **NOT using**:

    - Third-party antivirus programs (uninstall them)
    - Windows Defender (**disable**)
    - Windows SmartScreen (**disable**)
    - FaceIt Anti-Cheat
    - Riot Vanguard (Valorant Anti-Cheat)

    ### GPU Requirements

    **NVIDIA Users:**  Install NVIDIA App: [https://www.nvidia.com/en-us/software/nvidia-app/](https://www.nvidia.com/en-us/software/nvidia-app/)\
     Log in and update to the latest drivers\
     Enable **NVIDIA Overlay**

    **AMD Users:**  Install AMD Software: Adrenalin Edition (**NOT PRO**)\
     Update to the latest drivers\
     Enable **In-Game Overlay**\
     In Performance tab, enable **Performance Metrics Display**

    ### BIOS Requirement

    - Enter BIOS and disable **Intel Rapid Storage Technology**
    - Restart your PC again before continuing

  <Step title="Common Errors">
    ##  Common Errors & Fixes

    ###  Loader Closes Instantly

    - Reinstall: [https://aka.ms/vs/17/release/vc\_redist.x64.exe](https://aka.ms/vs/17/release/vc_redist.x64.exe)

    ### 🖱️ Menu Not Clickable

    - Run Steam as **non-administrator**
    - Ensure only NVIDIA/AMD overlays are enabled
    - Run overlay software as **administrator**

    ### 🧱 Visuals Not Working / Menu Disappears

    - Game may be patched → wait for update
    - Verify game files (full reinstall recommended)
    - Still not working → contact support

    ### 🛑 Antivirus Warning

    - Fully uninstall third-party antivirus (Windows 10/11)
    - Disable **Real-time Protection** and **SmartScreen**

    **Windows 11 Fix:**  Download: [https://disk.yandex.ru/d/5slbv4S2AL1M0w](https://disk.yandex.ru/d/5slbv4S2AL1M0w)\
     Run `Turn_ON.reg` → Restart\
     Run `Turn_OFF.reg` → Restart again
  </Step>
  <Step title="Advanced Errors">
    ##  BIOS / Hardware Errors

    ### Common Issues & Fixes

    - **AES instruction set not supported**\
      → CPU unsupported (contact support)
    - **Intel VT-X not enabled**\
      → Enable **Intel Virtualization Technology (VT-X)** in BIOS
    - **AMD-V not enabled**\
      → Enable **SVM Mode / AMD-V** in BIOS
    - **Secure Boot must be disabled**\
      → Disable Secure Boot in BIOS
    - **Unsupported firmware**\
      → Convert disk to GPT (mbr2gpt or disk utility)
    - **Secure Boot keys invalid**\
      → Enable Secure Boot → reboot → disable again
    - **VMX/SVM not supported**\
       Disable **Hyper-V** & **Virtual Machine Platform** in Windows Features\
       Run CMD as admin:\
      `bcdedit /set hypervisorlaunchtype off`\
       Restart PC
    - **Incorrect HWID**\
      Run in CMD (admin):
      ```text
      wmic diskdrive get Caption, SerialNumber
      wmic baseboard get SerialNumber
      ```
      - If commands fail → reinstall Windows
  </Step>
