## Common Errors

<AccordionGroup>
  <Accordion title="Loader Connection Issues?">
    1. Download: [Cloudflare Warp](https://one.one.one.one/)
    2. Connect to Warp then try to launch the loader!
  </Accordion>

  <Accordion title="Loader keeps wanting to restart PC?">
    Usual problems that causes this:

    1. Multiple Windows installs
    2. A second bugged EFI partition being around somewhere
    3. Some Laptops don't like it and in the restart process you will be stuck in a black screen (Rare case)

    If you have Nr. 1 physically unplug the other windows drive.

    If you have Nr. 2 then you can either erase the partition (needs IT knowledge) or do a windows reinstall.

    If you have Nr. 3 Contact Support
  </Accordion>

  <Accordion title="Crashing/Loader not starting">
    Run CMD as admin and run these commands seperately one by one: \
    `Dism /Online /Cleanup-Image /ScanHealth`

    `Dism /Online /Cleanup-Image /RestoreHealth`

    `sfc /scannow`
  </Accordion>

  <Accordion title="HVCI Issues">
    Launch CMD as Admin, type in these Commands: \
    `bcdedit /set hypervisorlaunchtype off`

    `"reg add "HKLM\SYSTEM\CurrentControlSet\Control\DeviceGuard\Scenarios\HypervisorEnforcedCodeIntegrity" /v "Enabled" /t REG_DWORD /d 0 /f"`

    `"reg add "HKLM\SYSTEM\CurrentControlSet\Control\DeviceGuard" /v EnableVirtualizationBasedSecurity /t REG_DWORD /d 1 /f"`
  </Accordion>
</AccordionGroup>
